Reminder project features

  • Add Event

    To add event - specify ALL it`s parameters in command line

    • http://Reminder/add?title=Java&day=25.05.2016&time=14.00

  • Display Event

    • Displays all available events

    http://Reminder/display

  • Search Event

    • To search for existing events use parameter as a search pattern

    http://Reminder/search?SomeValue

  • Remove Event

    • To remove event - specify it`s ID in parameters

    http://Reminder/remove?id=NUMBER
